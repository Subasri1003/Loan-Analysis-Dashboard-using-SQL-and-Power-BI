# Loan-Analysis-Dashboard-using-SQL-and-Power-BI

##Project Overview
The Loan Data Analysis Dashboard is an interactive Power BI project designed to analyze loan application data and uncover valuable insights related to loan performance, default risk, credit scores, educationa levels, income groups, and interest rates. The project utilizes SQL for data analysis and transformation and Poer BI for crating insightful and interactive dashboards. The dataset was imported from a CSV file and transformed into meaningful visual reports to support data driven decision-making 

##Tools Used
1- Power BI
2- SQL
3- CSV Dataset
4- Data Cleaning & Transformation
5- Data Visualization

##Key Insights
1- Created score plays a significant role in determining loan amounts.
2- Educational level influences borrowing patterns and loan requirements.
3- Certain loan purposes exhibit higher default risks.
4- Home ownership status impacts loan approval and repayment trends.
5- Interset rates vary based on loan intent and applicant characteristics.

##Process

3- Built data models and relationships.
4- Built interactive Power BI dashboards.
5- generated actionable business insights.
